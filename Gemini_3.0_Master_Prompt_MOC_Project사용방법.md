[프롬프트 사용 및 업데이트 가이드]
이 프롬프트는 단순한 질문-답변을 넘어, MOC 프로젝트의 구조를 이해하는 AI 개발자를 소환하는 도구입니다.

1. 초기 설정 (최초 1회):
위의 코드 블록 전체를 복사하여 Gemini 창에 붙여넣으십시오.
{{MINECRAFT_VERSION}}과 {{JAVA_VERSION}} 부분을 현재 프로젝트 환경에 맞게 수정하여 입력하면 더 좋습니다. (예: 1.20.1, Java 17)

2. 일상적인 작업 요청 예시:
새 능력 추가: "시간을 5초간 멈추는 'TimeStop' 능력을 impl 패키지에 추가해줘. 쿨타임은 60초야."
버그 수정: "게임이 끝났는데 아레나 블록이 복구되지 않아. ClearManager와 GameManager를 확인해줘."
코드 리팩토링: "AbilityManager의 등록 로직이 너무 길어. Reflection을 사용해서 자동으로 등록하게 바꿔줘."

3. 프로젝트 구조가 변경되었을 때:
만약 새 패키지(예: database)를 추가했다면, 프롬프트의 [C] Context & Mission 섹션 내 1. Project Anatomy 부분을 수정한 후 Gemini에게 "프로젝트 구조가 업데이트되었어. 이 새로운 프롬프트 정의를 기억해."라고 다시 입력해 주십시오.
